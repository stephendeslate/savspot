# Contract Templates

Contract templates let you create reusable contract structures that save time and ensure consistency across your client agreements. Define your terms once, then generate contracts for individual clients in seconds.

## Creating a Template

1. Navigate to `/contracts` and click **Templates**.
2. Click **New Template** to start from scratch.
3. Enter a template name that describes its purpose, such as "Standard Service Agreement" or "Wedding Photography Contract."
4. Write or paste your contract content into the editor.
5. Insert template variables where client-specific details will go.
6. Click **Save Template**.

## Template Variables

Template variables are placeholders that get replaced with actual values when you create a contract from the template. Use double curly braces to define them:

| Variable | Replaced With |
|----------|---------------|
| `{{client_name}}` | The client's full name |
| `{{client_email}}` | The client's email address |
| `{{service_name}}` | The name of the booked service |
| `{{service_price}}` | The price of the service |
| `{{start_date}}` | The contract or service start date |
| `{{end_date}}` | The contract or service end date |
| `{{business_name}}` | Your business name |

> **Tip:** Use descriptive variable names so that anyone on your team can understand what information each placeholder requires.

## Terms and Conditions Sections

Organize your templates with clearly labeled sections for readability and legal clarity:

1. **Scope of Services** — Define exactly what is included.
2. **Payment Terms** — Specify amounts, due dates, and accepted methods.
3. **Cancellation Policy** — Outline the cancellation and refund process.
4. **Liability** — State any limitations on liability.
5. **Confidentiality** — Include if handling sensitive client information.

## Using Templates for New Contracts

To create a contract from a template:

1. Go to `/contracts` and click **New Contract**.
2. Select a template from the dropdown.
3. Choose the client the contract is for.
4. Review and fill in any template variables that were not auto-populated.
5. Make any client-specific edits to the contract text.
6. Save and send the contract for signing.

> **Tip:** Keep a small library of focused templates rather than one massive catch-all. This makes it easier to select the right template and reduces the need for manual edits.
