# Quick Actions

Quick actions on the Dashboard let you perform common tasks without navigating away from your home page. They are designed for speed, reducing the number of clicks needed for everyday operations.

## Available Quick Actions

| Action | Description | Equivalent Navigation |
|--------|-------------|-----------------------|
| **New Booking** | Open the booking creation flow | `/bookings` > New Booking |
| **Add Client** | Create a new client profile | `/clients` > New Client |
| **Walk-In Booking** | Record a walk-in client immediately | `/bookings` > Walk-In |

## Using Quick Actions

### New Booking

1. Click **New Booking** on the dashboard
2. The booking creation dialog opens
3. Select a service, client, date, and time
4. Confirm the booking

This is the fastest way to schedule an appointment when a client calls or messages you. The flow is identical to creating a booking from the Bookings page.

### Add Client

1. Click **Add Client** on the dashboard
2. Enter the client's name, email, and phone number
3. Save the client profile

> **Tip:** You do not need to add a client separately before booking them. The booking creation flow lets you create a new client inline. Use this action when you want to add client details before they book.

### Walk-In Booking

1. Click **Walk-In Booking** on the dashboard
2. Select the service being provided
3. Assign or create the client
4. The booking is created for the current time with CONFIRMED status

Walk-in bookings skip the date and time selection since the client is already present. See [Walk-In Bookings](../booking-management/walk-in-bookings.md) for more details.

## Tips for Efficiency

> **Tip:** Quick actions are especially useful during busy periods when you need to create bookings or register clients rapidly without losing your place on the dashboard.

The quick action buttons are always visible at the top of the dashboard, so you can access them immediately after logging in or returning to the home page.
