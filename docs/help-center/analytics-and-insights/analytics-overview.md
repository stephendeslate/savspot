# Analytics Overview

The analytics page provides business performance metrics and reports. Navigate to **Analytics** (`/analytics`) to view your dashboard.

> **Note:** Analytics is restricted to **Admin** and **Owner** roles.

## KPI Cards

Six metric cards are displayed at the top of the page:

### Primary Metrics (4 cards)

| Card | Description |
|------|-------------|
| **Total Bookings** | All-time booking count |
| **Total Revenue** | All-time revenue |
| **Total Clients** | Unique clients served |
| **Avg Booking Value** | Average revenue per booking |

### Secondary Metrics (2 cards)

| Card | Description |
|------|-------------|
| **Completion Rate** | Percentage of bookings completed |
| **Cancellation Rate** | Percentage of bookings cancelled |

## Period Filter

Use the dropdown to filter data by time range:

- Last 7 days
- Last 30 days
- Last 90 days
- Last 12 months

## Revenue Trends

A table showing revenue and booking counts by period. This section requires a **Premium** subscription tier — an upgrade banner is shown if your plan does not include it.

| Column | Description |
|--------|-------------|
| **Period** | Time period |
| **Revenue** | Total revenue for the period |
| **Bookings** | Number of bookings |

## Staff Performance

A table showing performance metrics per team member. This section requires an **Enterprise** subscription tier.

| Column | Description |
|--------|-------------|
| **Staff Member** | Team member name |
| **Bookings** | Number of bookings handled |
| **Revenue** | Revenue generated |
| **Avg Rating** | Average review rating |
| **Completion Rate** | Percentage of bookings completed |

## Exporting Data

An **Export CSV** button is available but is not yet functional. CSV export is planned for a future release.

> **Tip:** Check your analytics regularly to identify trends in booking volume, revenue, and client growth.
