# Discounts

Create and manage discount codes that your clients can apply during the booking process. Discounts are configured at **Settings > Discounts** (`/settings/discounts`).

## Creating a Discount Code

1. Navigate to `/settings/discounts` and click **Create Discount**.
2. Enter a unique discount code (e.g., `SUMMER20`).
3. Select the discount type.
4. Set the discount value.
5. Configure optional restrictions.
6. Click **Save** to activate the discount.

## Discount Types

| Type | Description | Example |
|------|-------------|---------|
| Percentage | Reduces the price by a percentage of the service total | 20% off a $100 service = $80 |
| Fixed Amount | Reduces the price by a fixed dollar amount | $15 off a $100 service = $85 |

## Configuration Options

| Option | Description |
|--------|-------------|
| Usage Limit | Maximum number of times the code can be redeemed across all clients |
| Per-Client Limit | Maximum number of times a single client can use the code |
| Expiry Date | Date after which the code is no longer valid |
| Applicable Services | Restrict the discount to specific services, or apply to all |
| Minimum Booking Value | Minimum service price required for the discount to apply |

## Managing Existing Discounts

From the discounts list, you can:

- **Edit** a discount to change its value, limits, or expiry date.
- **Deactivate** a discount to temporarily stop it from being redeemed without deleting it.
- **Delete** a discount to remove it permanently.

> Tip: Use descriptive code names that are easy for clients to remember. Codes like `WELCOME10` or `HOLIDAY25` communicate the offer clearly.

> Tip: Set both a usage limit and an expiry date on promotional codes to prevent unexpected redemptions after a campaign ends.
