# Discounts

Create and manage discount codes for your clients at **Settings > Discounts** (`/settings/discounts`).

## Creating a Discount

1. Navigate to `/settings/discounts` and click **Create Discount**.
2. Fill in the discount details.
3. Click **Create Discount** to save.

## Discount Fields

| Field | Description |
|-------|-------------|
| **Code** | Unique discount code (e.g., `SUMMER20`) |
| **Type** | Percentage, Fixed Amount, or Free Hours |
| **Value** | The discount amount (percentage, dollar amount, or number of free hours) |
| **Application** | How the discount is applied (see below) |
| **Min Booking Amount** | Minimum booking value required for the discount to apply |
| **Max Uses** | Maximum number of times the code can be redeemed across all clients |
| **Valid From** | Start date for the discount |
| **Valid To** | Expiration date for the discount |

## Discount Types

| Type | Description | Example |
|------|-------------|---------|
| **Percentage** | Reduces the price by a percentage | 20% off a $100 service = $80 |
| **Fixed Amount** | Reduces the price by a flat amount | $15 off a $100 service = $85 |
| **Free Hours** | Waives charges for a number of hours | 2 free hours on an hourly service |

## Application Methods

| Method | Description |
|--------|-------------|
| **Code Required** | Client must enter the discount code during booking |
| **Automatic** | Discount is applied automatically when conditions are met |
| **Admin Only** | Only staff can apply the discount (not available to clients) |

## Managing Discounts

From the discounts list you can:

- **Edit** a discount to change its value, limits, or dates
- **Deactivate** a discount to stop it from being redeemed

> **Tip:** Use descriptive code names that are easy for clients to remember. Codes like `WELCOME10` or `HOLIDAY25` communicate the offer clearly.
