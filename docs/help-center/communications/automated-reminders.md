# Automated Reminders

SavSpot sends automated notifications for booking events. Configure which notifications are enabled at **Settings > Notification Preferences** (`/settings/notification-preferences`).

## Tenant-Level Notification Settings

Navigate to `/settings/notification-preferences` to configure which automated messages your business sends.

### Email Notifications

| Toggle | Description |
|--------|-------------|
| **Booking Confirmation** | When a new booking is confirmed |
| **Booking Reminder** | Upcoming booking reminders |
| **Booking Cancellation** | When a booking is cancelled |
| **Payment Received** | When a payment is processed |
| **Review Received** | When a client leaves a review |

### Push Notifications

| Toggle | Description |
|--------|-------------|
| **New Booking** | Push alert for new bookings |
| **Cancellation** | Push alert for cancellations |
| **Payment** | Push alert for payments |

### SMS Notifications

| Toggle | Description |
|--------|-------------|
| **SMS Reminder** | Send SMS reminders to clients (off by default) |

### Digest Settings

Bundle notifications into periodic summaries instead of receiving them individually:

1. Enable the **Digest** toggle.
2. Set the **Frequency**: Daily or Weekly.
3. Set the delivery **Time**.
4. For Weekly, select the **Day of Week**.

Click **Save Preferences** to apply changes.

> **Tip:** Enable SMS reminders to reduce no-shows. SMS has higher open rates than email for time-sensitive appointment reminders.
