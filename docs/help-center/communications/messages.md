# Messages

Communicate with your clients directly from SavSpot. Navigate to **Messages** (`/messages`) to view and manage conversations.

## Message Interface

The messages page uses a two-panel layout:

- **Left panel** — Thread list with search. Each thread shows the subject, client name, date, and an unread count badge.
- **Right panel** — Selected thread with message history and a reply input.

## Starting a New Conversation

1. Click **New Thread** (plus icon).
2. Fill in the dialog fields:

| Field | Description |
|-------|-------------|
| **Client Name** | Name of the client |
| **Subject** | Conversation subject line |
| **Message** | Initial message content |

3. Click **Create Thread**.

## Sending Messages

In an open thread:

1. Type your message in the text area at the bottom.
2. Press **Enter** to send (Shift+Enter for a new line).
3. Or click the **Send** button.

## Message Types

Messages are styled differently based on the sender:

| Type | Display |
|------|---------|
| **Staff** | Right-aligned, primary color background |
| **Client** | Left-aligned, accent color background |
| **System** | Centered, muted italic text |

## Searching Threads

Use the search input at the top of the thread list to filter conversations by subject or client name.

> **Tip:** Use messages for appointment-specific communication and follow-ups. For recurring communications like booking confirmations, set up automated templates at Settings > Communications instead.
