# Client Notes

SavSpot provides two note systems on each client profile: a general notes field and a structured notes panel. Both are visible only to your staff — clients never see them.

## General Notes

The general notes field is a simple textarea on the client profile's right sidebar.

1. Navigate to the client's profile at **Clients** > click the client's card.
2. Find the **Notes** card on the right side.
3. Type your note in the textarea.
4. Click **Save Notes** (the button is disabled until you make a change).

General notes are a single freeform text field. They are ideal for quick, running notes about a client's preferences or history.

## Structured Notes Panel

Below the general notes, the **Notes** panel provides a more organized system for adding, pinning, editing, and deleting individual note entries.

### Adding a Note

1. Expand the Notes panel (click the header with the sticky-note icon).
2. Type your note in the input field.
3. Submit to create a new note entry.

Each note is timestamped and associated with the staff member who created it.

### Managing Notes

| Action | How |
|--------|-----|
| **Pin a note** | Click the pin icon to keep a note at the top of the list |
| **Edit a note** | Click the edit button to modify the text |
| **Delete a note** | Click the delete button. This action cannot be undone |

Notes are displayed with pinned notes first, then by newest first.

## When to Use Each

- **General notes** — Quick, single-field summary (e.g., "Prefers morning appointments, allergic to latex")
- **Structured notes** — Timestamped entries from multiple team members (e.g., visit-by-visit observations)

> **Tip:** Keep notes professional and factual. All team members with access to the client directory can read them.
