# @savspot/ee — Enterprise Edition

This package contains SavSpot's Enterprise Edition features. These are premium features available to customers with a valid SavSpot Enterprise subscription.

## What's included

Enterprise features include: audit logging, workflow automation, digital contracts, quotes, custom domains, multi-location management, partner programs, AI recommendations, voice receptionist, accounting integrations, directory listing, platform admin, AI operations, and the public API.

## How it works

- **Self-hosted (AGPL core)**: All core booking features are free and open source.
- **Enterprise features**: Require a license key, set via `SAVSPOT_LICENSE_KEY` environment variable or per-tenant in the database.
- **Managed cloud**: License keys are provisioned automatically with your subscription.

## Getting a license

Enterprise subscriptions are available at [savspot.co/pricing](https://savspot.co/pricing) or by contacting hello@savspot.co.
